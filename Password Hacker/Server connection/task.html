Description

Let's imagine an admin who runs a website on the Internet. The site is becoming very popular, a large number of people register on it. They fill in their profiles, leave information there that is not intended to be public (for example, information about their credit cards). 

But the admin has completely forgotten about the security of his site. Now you can log in to the site with admin privileges without even having a login and password!

The first task in this project is to go to the admin's site, and the site will immediately give out all the secret information. Remember, as soon as you enter the site as an admin, it will automatically mean that you have obtained all the private data of the site. Therefore, the tasks of all other stages of the project will be to crack the admin password. At each stage it will be harder and harder to do. Good luck!

Your program should connect to the server using an IP address and a port from the command line arguments. You can parse them if you import ‘sys’ module to your program: see on an example how to use it.

You can use socket module to create this program. If you already know what socket means in computer science, you may use this example to build your socket program in python. If you don’t know anything about networking and inter-process communication, there is no need to panic: simply consult this comprehensive article to understand the topic.

Objectives

Your program will receive command line arguments in this order:

  1. IP address
  2. port
  3. message for sending

The algorithm is the following:

  1. Create a new socket.
  2. Connect to a host and a port using the socket.
  3. Send a message from the third command line argument to the host using the socket.
  4. Receive the server’s response.
  5. Print the server’s response.
  6. Close the socket.

Examples

The greater-than symbol followed by space (> ) represents the user input. Notice that it's not the part of the input.

Example 1:

> python hack.py localhost 9090 password
Wrong password!

Example 2:

> python hack.py 127.0.0.1 9090 qwerty
Connection Success!

Hint

To do this stage you should understand that the process of server-client communication is only the data exchange that computers can understand.  So it's only some arrays of bytes that are divided in some packets, frames, different sections of headers and blocks of our data that we need to use. It can be the text information, images or audios but it is still the byte array. So to send some information to the server we need to transform it into so called byte-arrays. When we recieve some information we should transform it into clear format (in this stage it's text). There are special functions : encode() and decode() that can transform strings into byte-arrays and backwards. To discover more about these functions, encoding codecs, python unicode support read this article on the official python web site. If you only want to know how to use these functions in practice use this example.
